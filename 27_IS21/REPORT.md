# Отчет по практической работе №1 (вариант 27)

## Название проекта
Каталог книг

## Реализованные модули
- `27_modulA.py` - хранение данных о книге (автор, название, год).
- `27_modulB.py` - поиск книг по автору или названию.
- `27_modulC.py` - случайная рекомендация книги из каталога.

## Файлы проекта
- `.gitignore`
- `requirements.txt`
- `main.py`
- `README.md`

- `27_modulB.py`
